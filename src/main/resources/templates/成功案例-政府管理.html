  • 国家电子政务外网移动安全接入管理平台项目

    国家电子政务外网移动安全接入管理平台是在国家信息中心指导下根据国家对移动信息安全及各省、市移动政务发展的要求而形成的具备前端安全接入、后端安全管理的综合性解决方案，当前已在国家信息中心、四川省政府、普洱市人民政府、广西省政府完成了三级部署及试点。

    主要功能：

    1. 1、设备安全接入政务外网
    2. 2、设备的统一管理
    3. 3、政务APP的推送
    4. 4、文档消息通讯录的下发推送

  • 国家公共资源交易服务平台移动信息推送支撑项目

    国家公共资源交易服务平台是一个“安全、高效、可靠”的政务信息共享交换公共基础设施。该平台实现全国公共资源交易信息的共享和发布、市场主体和专家信息的互认共享、市场主体信用信息和专家信用信息的交换共享，强化公共资源交易信息深度分析，为市场主体、社会公众和政府部门提供形式丰富的信息服务，为政府部门推进简政放权、优化市场环境、促进公平竞争、创新监管手段、提高公共资源配置效率提供便捷安全的电子政务支撑平台和决策依据，支持“实现平台之外无交易的目标”。

    主要功能：

    1. 1、用户根据不同需求订阅不同公告信息
    2. 2、政府采购信息公示时需要推送通知信息
    3. 3、用户需要及时的获取准确的招标信息
    4. 4、移动共享信息的安全与管理需求

  • 国家重大建设项目库系统

    国家重大建设项目库系统将实现对国家重大建设项目信息标准化采集、统一整理、信息调度等核心功能，实现数据即时采集、自动接收、汇总，数据经初步整合后形成实时更新的国家重大建设项目库，为领导决策、信息监测等提供技术支撑，实现重大建设项目网上调度。为满足国家重大建设项目库系统管理人员和用户移动化办公的需要，让管理人员和用户摆脱时间和空间的束缚，可在任何时间、任何地点处理与业务相关的工作，提高办公及管理的办事效率。因此，国家重大建设项目库系统远程管理平台在政务外网安全接入平台的基础上建设，开发重大库远程管理平台PC客户端、移动设备客户端、局域网客户端，满足管理人员和用户办公需要。

    主要功能：

    1. 1、满足国家信息安全主管部门对移动安全应用的要求
    2. 2、满足用户管理需求
    3. 3、提供SSL-VPN安全接入和SSH远程连接功能
    4. 4、满足保障数据资源安全需求
    5. 5、实现消息数据推送功能
